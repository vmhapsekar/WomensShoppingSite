## Install

## [Dependencies](id:dependencies)
- git
- [nvm](http://github.com/creationix/nvm)  
- [yarn](http://yarnpkg.com/) 

## [Start app](id:start-app)

1. Switch to the correct version of node: `nvm use`
2. Install all node modules: `yarn link-modules`
3. `yarn start`
